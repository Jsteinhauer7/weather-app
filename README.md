# Weather App

A web app that fetches and displays live weather data for any city using a free API.

## Features
- Search any city worldwide
- Displays temperature in Fahrenheit and Celsius
- Shows weather condition with animated emoji
- Wind speed display
- Handles connection errors gracefully

## Technologies
- Python, Flask
- Open-Meteo API (free, no API key required)
- HTML, CSS, Bootstrap 5

## How to Run
1. Install dependencies: `pip install flask requests`
2. Run: `python app.py`
3. Visit: `http://127.0.0.1:5000`
